Basically, we have 5 colors to draw a diagram about project "Client service"

Blue for inputting (clients import the personal data)

Green for outputting (for the web show the demand,announcement)

Blue and Green for Frontend

Yellow for assigning (calculate, assign definition, declare)

Red for logic, if, decision, true or false (sacrifice condition or not)

Orange for loop and controlling, while or for ( do the same tasks)

Yellow, Red, Orange for Backend
